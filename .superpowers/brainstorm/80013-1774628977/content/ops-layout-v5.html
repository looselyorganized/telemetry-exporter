V5 Changes

  • Header reordered — [Stats] [Models] [———spacer———] [Today 7D 30D All] [🔔 Bell]. Date removed, time range takes its spot on the right.
  • Notification bell dropdown — Click reveals list of recent alerts with timestamps, dismiss (✕) per item, "Mark all read", and "See all notifications" link at bottom.
  • 2-row stacking grid — Uses CSS grid-template-rows: 1fr 1fr; grid-auto-flow: column. Cards fill column-first: top-left → bottom-left → top-right → etc. Scrolls horizontally when more than ~3 columns. On narrow screens, collapses to grid-template-rows: 1fr (single row).
  • "+" card spans both rows — Tall thin add button at the end of the scroll.
  • Active cards are taller (they have the Agents box), idle cards are shorter — the 2-row grid handles this naturally since row height adapts to the tallest card.
